# Volt SQL Editor

Web based sql editor for VoltDB

## Getting Started

Download or clone the project
```
git clone https://github.com/aamadeo27/vsqle.git
```

### Prerequisites
Node v8 or later

### Installing

```
cd client/
npm install
npm run build
cd server/
npm install
```
